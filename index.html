About Me

I am a full-stack software developer based in Kirkland, Washington. The following is an overview of some of my more recent projects.

Team Projects

Good Watches is a full stack web-based app written in Node.js aimed at helping to find movies and/or TV shows to watch. It uses React Redux for its front end, Mongoose to manage its MongoDB back end, Express to set up RESTful API routes, Passport for handling authorization through Google or Twitter and a variety of other technologies to link everything together.

Something Borrowed is a full stack web application written in Node.js using Express and Handlebars for server-side rendering, Passport for authorization, and Sequelize for database management.

Kite Club is a client-side JavaScript tool to assist in finding the best place to find a kite. It uses Materialize for UI components, Dark Sky API for weather data, and OpenLayers for map rendering.

Solo Projects

Camels and Silk is an open source PC game currently in development. It is written in C++ and uses SDL for 2d graphics, sqlite3 for managing game data, Google FlatBuffers for saving game states, and Boost for game logic and file system access across multiple platforms.

Lizard Spock is an online implementation of an expanded Rock Paper Scissors game which uses Google Firebase to establish real-time connections between players. It uses bootstrap and font awesome for its grid functionality and button themes.

Star Wars RPG is a browser-based role playing game written using jQuery.

Chess Trivia is a multiple choice chess quiz written using jQuery.

Shakespeare Guess is a hang-man style game written in plain, vanilla JavaScript using CSS and HTML to display content.
